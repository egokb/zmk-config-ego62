# 📖 Ego62 设置与自定义教程

[🇺🇸 View English Version](./Ego62_tutorial-EN.md)

本教程提供两种自定义 Ego62 键盘的方法：
- **方式一**：ZMK Studio（最简单，实时编辑）
- **方式二**：GitHub + 键位编辑器（高级，更多控制）

---

## ⚡ 方式一：ZMK Studio（推荐）

最简单的键位自定义方式，支持实时预览。

### 前置条件
- ✅ 已刷写支持 ZMK Studio 的固件
- ✅ USB-C 数据线连接到接收器
- ✅ 现代网页浏览器（Chrome、Edge、Firefox）

---

### 第一步：连接键盘

1. 使用 USB-C 数据线将**接收器**连接到电脑
2. 确保左右手键盘已开机
3. 等待键盘连接（约 5 秒）

---

### 第二步：打开 ZMK Studio

1. 打开浏览器
2. 访问 [https://zmk.studio/](https://zmk.studio/)
3. 网站应自动检测到你的键盘

---

### 第三步：解锁键盘

1. 在 Ego62 上切换到 **System 层**（第 2 层）：
   - 按住切层键（默认：底行）
   - 或使用切层切换键（如果已配置）
2. 找到并按下 `&studio_unlock` 键
3. 键盘已解锁，可以进行编辑

> 💡 **提示**：解锁键通常放在 System 层以确保安全，防止误触修改键位。

---

### 第四步：编辑键位

1. **可视化编辑器**：点击布局图中的任意按键
2. **选择行为**：从下拉菜单中选择：
   - 按键（普通按键）
   - 层控制（mo、to 等）
   - 蓝牙控制
   - 媒体键
   - 更多...
3. **保存更改**：点击"保存"按钮将更改写入键盘

---

### 第五步：测试更改

1. 使用新的键位进行输入测试
2. 更改即时生效
3. 如果不满意，随时可以恢复默认设置

---

### 恢复默认设置

如果你想恢复到原来的键位：
1. 在 ZMK Studio 中找到**"Restore Stock Settings"（恢复出厂设置）**按钮
2. 点击恢复到固件默认键位

---

## 🛠️ 方式二：GitHub + 键位编辑器

适合需要版本控制的高级自定义。

---

### 第一部分：Fork 仓库

#### 步骤 1：创建 GitHub 账号

1. 访问 [GitHub](https://github.com)
2. 注册或登录你的账号

#### 步骤 2：Fork 仓库

1. 访问 [Ego62 固件仓库](https://github.com/egokb/zmk-config-ego62)
2. 点击右上角的 **"Fork"** 按钮
3. 等待 Fork 完成（你会被重定向到你的副本）

---

### 第二部分：启用 GitHub Actions

1. 在你 Fork 的仓库中，点击 **Actions** 标签页
2. 点击绿色的 **"I understand my workflows, go ahead and enable them"** 按钮
3. GitHub Actions 现在已启用，会自动编译固件

---

### 第三部分：编辑键位

#### 步骤 1：打开键位编辑器

1. 访问 [Nick Coutsos 键位编辑器](https://nickcoutsos.github.io/keymap-editor/)
2. 点击 **"Authorize with GitHub"**（使用 GitHub 授权）
3. 选择你 Fork 的仓库
4. 授予必要的权限

#### 步骤 2：自定义按键

1. 点击可视化布局中的任意按键
2. 从下拉菜单中选择所需的行为
3. 重复操作修改所有需要的按键

#### 步骤 3：保存更改

1. 点击 **"Save"**（保存）按钮
2. 输入提交信息（或使用默认信息）
3. 点击 **"Commit changes"**（提交更改）
4. 等待提交完成

---

### 第四部分：下载固件

#### 步骤 1：等待构建

1. 回到你的 GitHub 仓库
2. 点击 **Actions** 标签页
3. 你应该看到新的工作流正在运行
4. 等待约 5 分钟让构建完成

#### 步骤 2：下载固件

1. 点击已完成的工作流运行
2. 向下滚动到 **Artifacts**（构建产物）部分
3. 下载 `firmware.zip`
4. 解压 ZIP 文件获取 `.uf2` 文件

---

### 第五部分：刷写固件

⚠️ **重要**：按此顺序刷写以避免连接问题！

#### 刷写顺序：
1. `ego62_left.uf2`（左手）
2. `ego62_right.uf2`（右手）
3. `ego62_dongle.uf2`（接收器 - **最后！**）

#### 每个设备的刷写步骤：

1. **进入刷机模式**：双击设备背面的重置按钮
2. **复制文件**：将 `.uf2` 文件拖放到出现的 USB 磁盘中
3. **等待**：磁盘会消失，设备自动重启（约 5 秒）
4. **下一个设备**：重复上述步骤刷写下一个设备

---

## 🔄 后续更新

初次设置完成后，更新键位更简单：

### 使用 ZMK Studio：
1. 通过 USB 连接接收器
2. 打开 ZMK Studio
3. 编辑并保存 - 完成！

### 使用 GitHub 方式：
1. 打开键位编辑器
2. 修改并保存
3. 从 Actions 下载新固件
4. 刷写三个设备
